# 🧮 Interactive Calculator (Tkinter GUI)

A simple interactive calculator built using Python and Tkinter.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero error handling
- Clear button
- GUI number buttons(0–9)

## 🛠 Technologies Used
- Python
- Tkinter (GUI)

## 📦 How to Run the program

1. Clone the repository
2. Open the folder in VS Code
3. Run:

```bash
python calculator.py
